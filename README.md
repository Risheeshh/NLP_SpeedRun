# NLP_SpeedRun
Sentiment Analysis, Similarity Searches, NER, Intent-based chatbot
Speedrunning thru all of the above. Using manually curated amazon product reviews to make a "PRODUCT REVIEWS SENTIMENT ANALYSIS" for Q1, a glove (from spaCy) approach for Word similarity, NLTK for NER on sample news snippets and 12-vector space matrices escribing explicit NLP/AI principles securely properly for the chatbot.

Simplistic code for beginner learner getting used to these technology
